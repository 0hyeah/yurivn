Help Index > Add New Itemtype
Go to your AdminCP, click Item Types Managements under kBank Item, if this is the first time you do so, an Add Item Type will show up!


Included inside kBank 2.0 is 4 itemtypes: announce, pack, customize_userinfo and profile_images (you will find out their abilities later)


After filling the Add Item Type form (with Name, Filename, Description, Price,.v.v.) you will be redirected to edit the new itemtype settings (click here for detail information).
You have 3 itemtypes now:


We have 4 itemtypes so what is the rest? It's pack, a special itemtype! It has no real ability but it is a package that contains others! Choose to add new itemtype and select pack


In Edit Settings page, there are 2 settings:

  • Itemtype(s): Enter itemtypeid of itemtypes that you want to be included. If you want this pack contains 2 or more items of the same itemtype just repeat it's itemtypeid. For example: "1,2,2,3,3,3"
  • Duration Step Price: All itemtypes included will have the same active duration as the pack

You are done!
