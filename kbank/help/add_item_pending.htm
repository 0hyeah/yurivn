Help Index > Add New Item > Approve a Pending Item
To require more than 1 kBank Administrator permission to add new item. You must have more than 1 kBan kAdministrator


Access kBank Settings and edit Admin(ID)s value


Access kBank Item Settings and edit Add New Item Approval value


That's it! Now, when trying to add new item, it will not be shown in Shop! It's status is Pending until other kBank Administrator approves


Other kBank Administrators will see a notices if there are some pending items


He/she can click Approve like this to allow item to be sold or do nothing


After enough approvals, item will be sold as regular (or bidding if the Auction mode had been enabled)
