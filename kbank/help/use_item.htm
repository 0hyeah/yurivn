With each itemtype have a little difference using items but there are 2 types of item action: Enable/Disable and Use. Item action only can be done by item owner in My Items page

Announce
Click Enable to use item


In the next form, enter URL and text to display then click Enable Now


A small notice about item will display (My Items page). Click Edit Option to change URL,text. Click Disable Item to disable showing links


Links will be shown in Sponsored Links section in random positions. Maximum Announcement Entries shown can edit in kBank Item Settings



Customize Userinfo
Click Enable to use item


Depend on itemtype's settings specified by kBank Administrator, the input form can be different. Enter decied username in Username field. Select a color in Username Color. Select Yes in username Strong to display username in bold state. The same for Usertitle's


After pushing Enable Now, customized username/usertitle will show up almost everywhere (My Items, Member profile, Who's Online, Post view,.v.v.)



Profile Images
Click Use Item to use item


This is default permission of Registered Users usergroup


After using Profile Images, permission changed!

Edit Signature

Note: All item have an active duration and it will expire after that period of time

With Profile Images, when item expired, member's avatar and signature picture will be resized or removed


Member will also receive a Private Message



Pack
Click Un-Pack to use item


Included items will be extracted to your inventory (My Items) so you can use/sell/gift them as you like
